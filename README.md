# To know

This program is not an extension for a browser so it won't interact
with the browser. 

# Install

## Windows

```
Simply download TShark or Wireshark.
```

## Linux
```
Download TShark and potentially everything else will be fine.
If that's not enough you have to have wireshark in your user's
group.
```
